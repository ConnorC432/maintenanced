# maintenanced
This program is inspired by the Steam Deck's ability to download games while in a low power mode, before going to sleep. <br>
Details on how this works can be found [here](https://steamcommunity.com/games/1675200/announcements/detail/771930569635267985) <br>
This program aims to download/update anything required in a low power state before shutting down.
