# Contributing
Contributions to this project are welcome!

## Getting Started
Clone your fork, or click [here](https://git.anorlondo.dev/AnorLondo/maintenanced/-/forks/new)
```
git clone https://git.anorlondo.dev/AnorLondo/maintenanced.git
```

## Code Style
This project follows the Google C++ Style Guide. Details can be found [here](https://google.github.io/styleguide/cppguide.html)
